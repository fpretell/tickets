# Frontend React for api saga

**Description:** Frontend React for Api Users

## Enviroment

    SO: Linux (Ubuntu) recommended
    NodeJs: v10.21.0
    Npm: 6.14.4

## Installation

Clone project 
```bash
git clone https://github.com/fpretell/frontendsaga.git
```

Then, inside node_server
```bash
npm install
```

To start server
```bash
npm start
```
